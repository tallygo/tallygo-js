## Installation

### From NPM

```
npm install tallygo-kit
yarn add tallygo-kit
```


## Development


### Set up:

```
yarn install
```

### Lint and test:

```
yarn test
```

### Test:

```
yarn test:unit
```

### Build

```
yarn build:dev
```
generates files in `/dist` directory.

### Dev Server
```
yarn start
```

Navigate to: [http://localhost:9966/examples/](http://localhost:9966/examples/)
